# SRAM variable locations
|Offset|Description|
|------|-----------|
| 0x24 |Workmode, determined by chip model.(0x90 : Serial, 0x10 : Printer, 0xD0 : MIDI)|
| 0x3c |VID low    |
| 0x3d |VID high   |
| 0x3e |PID low    |
| 0x3f |PID high   |
